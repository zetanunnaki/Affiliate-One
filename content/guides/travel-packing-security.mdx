## The Security Travel Kit

Before you travel, pack these security essentials alongside your clothes and chargers. The total cost is under $100 and could save you from a devastating data breach.

## Hardware Kit

| Item | Cost | Why |
|------|------|-----|
| USB data blocker | $5-10 | Blocks juice jacking at public USB charging stations |
| Privacy screen filter | $25-40 | Prevents shoulder surfing on planes, cafes, trains |
| Portable battery pack (20,000mAh) | $30-50 | Avoid public USB charging entirely |
| Cable lock (Kensington) | $15-25 | Secure laptop at co-working spaces and hotels |
| YubiKey (optional) | $25-55 | Hardware 2FA for high-value accounts |
| Travel router (GL.iNet) | $30-80 | Personal VPN-enabled Wi-Fi anywhere |

## Software Kit (Install Before Departure)

- **VPN app** on all devices (download before visiting restrictive countries)
- **Password manager** synced across devices
- **Authenticator app** with cloud backup (Authy)
- **Signal** for encrypted messaging
- **Offline maps** downloaded for your destination
- **Translation app** downloaded for offline use
- **Find My Device** enabled on all devices

## Documentation Kit

- **Recovery codes** for all 2FA accounts (stored in password manager + printed)
- **Emergency contacts** (embassy, bank international number, travel insurance)
- **Device serial numbers** (for police reports if stolen)
- **Travel insurance policy number**
- **VPN manual configuration files** (backup if app doesn't work)

## Pre-Departure Checklist

1. All software updated to latest versions
2. Full-disk encryption verified (BitLocker/FileVault)
3. VPN installed and tested
4. Cloud backup completed
5. 2FA backup codes accessible
6. Bank notified of travel dates
7. Destination internet restrictions researched
8. eSIM or local SIM arranged
